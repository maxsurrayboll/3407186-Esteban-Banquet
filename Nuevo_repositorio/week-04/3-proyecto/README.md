# 💊 Generador de Mensajes - Sistema de Gestión de Farmacias

## 📌 Descripción
Este proyecto consiste en un generador de mensajes en consola desarrollado en JavaScript. Está aplicado a un sistema de gestión de farmacias, permitiendo mostrar información de medicamentos, validar datos y generar notificaciones dinámicas utilizando métodos de strings y template literals.

---

## 🎯 Objetivo
Construir un sistema que genere fichas informativas de medicamentos en inventario, aplicando transformaciones de texto y validaciones sobre los datos.

---

## ⚙️ Tecnologías utilizadas

- JavaScript  
- Node.js  

---

## 🚀 Cómo ejecutar el proyecto

1. Tener instalado Node.js  
2. Descargar o clonar el proyecto  
3. Abrir una terminal en la carpeta  
4. Ejecutar:

```bash
node script.js
