# 💊 Sistema de Gestión de Farmacia - Métodos de Arrays

## 📌 Descripción

En esta fase del desarrollo se implementan métodos avanzados de arrays en JavaScript, como `map`, `filter` y `reduce`, para optimizar el manejo de datos dentro del sistema de farmacia.

Estos métodos permiten trabajar de manera más eficiente y declarativa con la información del inventario.

---

## 🧠 Tecnologías utilizadas

- JavaScript
- Métodos de arrays
- Programación funcional
