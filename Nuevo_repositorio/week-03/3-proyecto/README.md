# 💊 Calculadora - Sistema de Gestión de Farmacias

## 📌 Descripción
Este proyecto consiste en una calculadora desarrollada en JavaScript aplicada a un sistema de gestión de farmacias. Permite simular la compra de medicamentos, calcular el total a pagar, aplicar descuentos y controlar el inventario de productos.

---

## 🎯 Funcionalidades

- 🧮 Cálculo del total de compra de medicamentos  
- 💸 Aplicación de descuentos según el monto total  
- 📦 Actualización del inventario  
- 🔍 Validación de productos agotados  
- 🔗 Uso de operadores lógicos para decisiones del sistema  

---

## ⚙️ Tecnologías utilizadas

- JavaScript  
- Node.js (para ejecutar el archivo)  

---

## 🚀 Cómo ejecutar el proyecto

1. Asegúrate de tener instalado Node.js  
2. Descarga o clona este repositorio  
3. Abre una terminal en la carpeta del proyecto  
4. Ejecuta el siguiente comando:

```bash
node script.js
