# Semana 1 - Introducción a JavaScript

## Proyecto
Sistema de Gestión de Farmacias

## Descripción
En esta primera semana se trabajó con los conceptos básicos de JavaScript, utilizando el entorno de ejecución Node.js para mostrar información en la consola.

El objetivo fue crear un programa que represente una entidad del dominio del sistema mediante el uso de diferentes tipos de datos.

## Entidad seleccionada
Medicamento

## Información del medicamento

- Nombre: Acetaminofén
- Código: MED-001
- Descripción: Analgésico que ayuda a aliviar el dolor y reducir la fiebre.
- Precio: 2500 COP
- Cantidad en inventario: 120 unidades
- Requiere fórmula médica: No
- Disponible en inventario: Sí
